What is AJAX and what is it used for?

Asynchronous Javascript And XML. It is used to update the html in the background without disrupting the user experience or design of the html page. It is a way to communicate with a server without refreshing the current webpage.

What is JSON and what is it used for?

JSON stands for JavaScript Object Notation. It is used for ease of reading and writing.

What does 'jqXhr' represent in the curriculum AJAX examples?

jqXhr is the AJAX request object.

What's the difference between $.ajax() and $.get()?

With the .ajax() method you can specifiy what type of request you are trying to make, such as: post, get, put, or delete. The .get() method is shorthand for the get request.

Is jQuery required to make AJAX requests?

JQuery is not necessary to make AJAX requests, but jQuery it simplifies the syntax for AJAX requests.
